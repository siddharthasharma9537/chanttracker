================================================================================
ARCHITECTURE OVERVIEW
================================================================================

Navigation Structure:
  Desktop (lg+): Fixed left sidebar (w-64) + 4 nav items + user card
  Mobile (<lg): Fixed bottom navbar (4 items) + icons + labels

Layout Components:
  - MainLayout.tsx: Wraps protected screens (header + nav)
  - Navigation.tsx: Responsive nav (sidebar + bottom bar)
  - Header.tsx: App header with settings & sign out

Auth System:
  - Providers.tsx: Initializes auth on app mount
  - useAuth.ts: Auth methods (signIn, signUp, signOut)
  - authStore.ts: Auth state (Zustand)
  - Protected screens use auth guard pattern (useEffect redirect)

Data Flow:
  Dashboard: useDashboard() → get_today_progress RPC
  Chant: useStartSession() → startSession RPC, then local counter
  History: useSessions() → SELECT sessions query
  Settings: useForm + upsert to profiles table

Offline Support:
  offlineStore: Tracks online status, queues pending sessions
  Auto-sync when reconnected to internet
