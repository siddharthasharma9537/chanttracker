💡 KEY DECISIONS:

  ✅ Monorepo structure (single source of truth)
  ✅ Supabase backend (replaces FastAPI + JWT + bcrypt)
  ✅ Shared API client (@chanttracker/api)
  ✅ RLS-based auth (no custom auth layer)
  ✅ Cloudflare edge (KV + R2 + D1)
  ✅ GitHub Actions CI/CD (auto-deploy on push)
  ✅ Native iOS (Swift + widget)
  ✅ Native Android (Kotlin Compose)
  ✅ PWA web (Next.js + installable)
