================================================================================
DEEP ROUTE PROTECTION PATTERN
================================================================================

Every protected screen (/dashboard, /chant, /history, /settings) follows:

  ┌─────────────────────────────────────┐
  │ Page Component Mounts               │
  └────────────┬────────────────────────┘
               │
               ↓
  ┌─────────────────────────────────────┐
  │ useAuth() hook                      │
  │ ├─ user from authStore              │
  │ ├─ isLoading from authStore         │
  │ └─ isSignedIn = !!user              │
  └────────────┬────────────────────────┘
               │
               ↓
  ┌─────────────────────────────────────┐
  │ useEffect(() => {                   │
  │   if (!authLoading && !isSignedIn)  │
  │     router.push('/auth/signin')     │
  │ })                                  │
  └────────────┬────────────────────────┘
               │
      ┌────────┴────────┐
      │                 │
 LOADING            NOT LOADING
      │                 │
      ↓                 ↓
  ┌────────────┐    ┌─────────────────┐
  │ Return     │    │ Check isSignedIn│
  │ <Spinner>  │    └────────┬─────────┘
  └────────────┘             │
                  ┌──────────┴──────────┐
                  │                     │
             TRUE                   FALSE
                  │                     │
                  ↓                     ↓
            ┌──────────┐      ┌──────────────────┐
            │ Render   │      │ Return null      │
            │ Page     │      │ (redirect in     │
            │ Content  │      │  progress)       │
            └──────────┘      └──────────────────┘


================================================================================
API CALLS & DATA FLOW
================================================================================

AUTHENTICATION
├─ Providers.tsx (on mount)
│  └─ supabase.auth.getUser()
│  └─ supabase.auth.onAuthStateChange()
├─ useAuth.signIn(email, password)
│  └─ supabase.auth.signInWithPassword()
├─ useAuth.signUp(email, password, displayName)
│  └─ supabase.auth.signUp()
└─ useAuth.signOut()
   └─ supabase.auth.signOut()

DASHBOARD DATA
├─ useDashboard() hook
│  └─ RPC: get_today_progress()
│     ├─ Returns: done, target, streak, total
│     └─ Refetch on user click

CHANT SESSION
├─ useStartSession() mutation
│  └─ RPC: start_session(mantra_id)
├─ useSessionCounter() hook (local state)
│  ├─ increment() / decrement()
│  ├─ pause() / resume()
│  ├─ complete()
│  │  └─ RPC: complete_session(session_id)
│  └─ abandon()
│     └─ RPC: mark_session_abandoned(session_id)

HISTORY
├─ useSessions() hook
│  └─ Query: SELECT * FROM sessions WHERE user_id = auth.uid()
│     ├─ Filter by date (client-side in SessionList)
│     └─ Show grouped by date

SETTINGS
├─ SettingsForm.tsx useEffect (on mount)
│  └─ Query: SELECT * FROM profiles WHERE id = user.id
├─ SettingsForm.onSubmit()
│  └─ UPSERT profiles table with new values
└─ applyTheme()
   └─ Sets HTML class (client-only, no API call)

OFFLINE SUPPORT
└─ offlineStore
   ├─ Track: navigator.onLine
   └─ Queue: pending sessions when offline
      └─ Sync: when reconnected

================================================================================
AUTH STATE LIFECYCLE
================================================================================

INITIAL LOAD
┌──────────────────────────┐
│ authStore.user = null    │
│ authStore.isLoading      │
│  = true                  │
└────────────┬─────────────┘
             │
             ↓
┌──────────────────────────┐
│ Providers.tsx useEffect  │
│ calls getUser()          │
└────────────┬─────────────┘
             │
      ┌──────┴──────┐
      │             │
   SUCCESS       ERROR
      │             │
      ↓             ↓
   ┌────┐      ┌────────────┐
   │Set │      │ setUser    │
   │user│      │ (null)     │
   └────┘      └────────────┘
      │             │
      └──────┬──────┘
             │
             ↓
   ┌──────────────────┐
   │ isLoading = false│
   │ Pages can now    │
   │ check isSignedIn │
   └──────────────────┘

SIGN IN SUCCESS
┌────────────────────┐
│ useAuth.signIn()   │
│ called             │
└────────┬───────────┘
         │
         ↓
┌────────────────────────────┐
│ supabase.auth.             │
│ signInWithPassword()       │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ JWT received from Supabase │
│ Stored in browser session  │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ setUser(data.user)         │
│ authStore.user updated     │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ All pages now see:         │
│ isSignedIn = true          │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ router.push('/dashboard')  │
│ Page renders, no redirect  │
└────────────────────────────┘

SIGN OUT
┌────────────────────┐
│ useAuth.signOut()  │
│ called             │
└────────┬───────────┘
         │
         ↓
┌────────────────────────────┐
│ supabase.auth.signOut()    │
│ - Invalidates JWT          │
│ - Clears session           │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ clearAuth()                │
│ authStore.user = null      │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ isSignedIn = false         │
│ All protected pages check  │
│ useEffect & redirect       │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ router.push('/auth/signin')│
│ (from SettingsForm or      │
│  any protected page guard) │
└────────────────────────────┘
